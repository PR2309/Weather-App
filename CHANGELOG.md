# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
- Add development server with Express.js
- Add alternative CSS styling options
- Improve project configuration files
- Enhanced documentation and setup instructions

## [1.1.0] - 2025-11-27
### Added
- Dynamic weather background animations based on weather conditions
- Improved responsive design for mobile devices
- Enhanced error handling for API failures
- Added keyboard shortcuts (Enter to search, Space to focus input)
- Unit conversion features for temperature, wind speed, and pressure

### Changed
- Updated UI animations and transitions
- Improved weather icon selection logic
- Enhanced accessibility with proper ARIA labels

### Fixed
- Fixed timezone calculations for local time display
- Corrected wind speed conversion formulas
- Improved error state display

## [1.0.0] - 2025-10-15
### Added
- Initial release of Weather App
- OpenWeather API integration
- Real-time weather data display
- City search functionality
- Weather condition icons and backgrounds
- Basic responsive design
- Progressive Web App (PWA) support

### Features
- Temperature display with unit selection (°C/°F/K)
- Humidity percentage
- Wind speed with multiple unit conversions
- Atmospheric pressure with hPa/mmHg conversion
- Local time display with 12/24-hour format
- Day/night phase detection
